# OpenSunstar v0.1.0 發布說明（繁體中文）

**發布日期：** 2026-06-19  
**授權條款：** MIT

---

## 概覽

OpenSunstar v0.1.0 是 **首次公開發布** — 面向 AI 程式設計 CLI 工具的原生桌面控制台。在一個應用程式中完成供應商切換、Agent 配置（MCP / Skills / Prompts）與多儲存庫專案組合洞察。

**下載：** [GitHub Releases](https://github.com/alisunstar/OpenSunstar/releases/tag/v0.1.0)

---

## 亮點

### 分鐘級接入

- **快速接入**精靈：供應商 → API 金鑰 → 套用到 CLI
- **50+ 供應商預設**，覆蓋七款支援工具
- **系統匣**快速切換；Claude Code 支援熱切換，無需重啟終端機

### 統一 Agent 工作區

- MCP 面板含探索註冊表與按應用同步
- Skills 支援 GitHub、ZIP、skills.sh、ClawHub、ModelScope
- Prompts、命令、鉤子、權限、Subagent、工作階段

### 專案組合與營運

- **專案組合儀表板** — 多 Git 儲存庫研發駕駛艙（非任務看板）
- 統一 **7 天提交指標** 用於卡片、矩陣與 AI 週報
- 用量儀表板、預算告警、WebDAV / S3 雲端同步

### 可靠性

- SQLite 儲存、原子寫入與自動備份
- 本機代理、自動故障轉移、熔斷器
- Windows · macOS · Linux（Tauri 2）

---

## 支援的工具

Claude Code · Claude Desktop · Codex · Gemini CLI · OpenCode · OpenClaw · Hermes

## 升級說明

這是首次發布，無需從舊版 OpenSunstar 遷移。首次啟動會將現有 CLI 設定匯入為 **default** 供應商。

## 已知限制（v0.1.0）

- 大多數 CLI 在切換供應商後需 **重新啟動終端機**（Claude Code 熱切換除外）。
- 專案組合階段 / MVP 進度存於 localStorage；SQLite 遷移計劃於後續版本。
- WebDAV/S3 自動同步失敗僅在設定頁提示，不會彈出全域通知。
