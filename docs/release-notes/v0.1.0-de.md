# OpenSunstar v0.1.0 Release Notes (Deutsch)

**Veröffentlichungsdatum:** 19.06.2026  
**Lizenz:** MIT

---

## Überblick

OpenSunstar v0.1.0 ist die **erste öffentliche Version** — eine native Desktop-Steuerzentrale für KI-Coding-CLI-Tools. Anbieterwechsel, Agent-Konfiguration (MCP / Skills / Prompts) und Multi-Repo-Portfolio-Einblicke in einer App.

**Download:** [GitHub Releases](https://github.com/alisunstar/OpenSunstar/releases/tag/v0.1.0)

---

## Highlights

### In Minuten verbinden

- **Simple Connect**-Assistent: Anbieter → API-Schlüssel → auf CLI anwenden
- **50+ Anbieter-Presets** für sieben unterstützte Tools
- **System-Tray**-Schnellwechsel; Claude Code Hot-Switch ohne Neustart

### Einheitlicher Agent-Arbeitsbereich

- MCP-Panel mit Discovery-Registry und Sync pro App
- Skills aus GitHub, ZIP, skills.sh, ClawHub, ModelScope
- Prompts, Befehle, Hooks, Berechtigungen, Subagents, Sitzungen

### Portfolio & Betrieb

- **Portfolio-Dashboard** — Multi-Git-Repo-Cockpit (kein Task-Kanban)
- Einheitliche **7-Tage-Commit-Metriken** für Karten, Matrix und KI-Wochenberichte
- Nutzungs-Dashboard, Budgetwarnungen, WebDAV / S3-Cloud-Sync

### Zuverlässigkeit

- SQLite-Speicher mit atomaren Schreibvorgängen und Auto-Backup
- Lokaler Proxy, Auto-Failover, Circuit Breaker
- Windows · macOS · Linux (Tauri 2)

## Upgrade-Hinweis

Dies ist die Erstveröffentlichung. Es gibt keine Migration von einer früheren OpenSunstar-Version. Bestehende CLI-Konfigurationen werden beim ersten Start als **default**-Anbieter importiert.

## Bekannte Einschränkungen (v0.1.0)

- Die meisten CLI-Tools benötigen nach dem Anbieterwechsel einen **Terminal-Neustart** (Ausnahme: Claude Code Hot-Switch).
- Portfolio-Phase / MVP-Fortschritt liegt in localStorage; SQLite-Migration für eine spätere Version geplant.
- Auto-Sync-Fehler bei WebDAV/S3 erscheinen in den Einstellungen, nicht als globale Benachrichtigungen.
