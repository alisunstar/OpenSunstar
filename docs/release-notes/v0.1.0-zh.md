# OpenSunstar v0.1.0 发布说明

**发布日期：** 2026-06-19  
**许可证：** MIT

---

## 概述

OpenSunstar **v0.1.0** 为**首次公开发布版本**——面向 AI 编程 CLI 的原生桌面控制台，统一管理供应商切换、Agent 配置（MCP / Skills / Prompts）与多仓库项目组合洞察。

**下载：** [GitHub Releases](https://github.com/alisunstar/OpenSunstar/releases/tag/v0.1.0)

---

## 版本亮点

### 快速接入

- **快速接入**向导：选供应商 → 保存 Key → 应用到 CLI
- **50+ 供应商预设**，覆盖七个支持的 CLI 工具
- **系统托盘**即时切换；Claude Code 支持热切换，无需重启

### 统一 Agent 工作区

- MCP 面板：发现注册表、按应用双向同步
- Skills：GitHub / ZIP / skills.sh / ClawHub / ModelScope 一键安装
- Prompts、命令、钩子、权限、Subagent、会话管理

### 项目组合与运维

- **项目组合仪表盘** — 多 Git 仓库驾驶舱（非拖拽任务看板）
- **7 天统一提交指标** — 总览卡片、矩阵、AI 周报共用同一窗口
- 用量仪表盘、预算告警、WebDAV / S3 云同步

### 可靠性与跨平台

- SQLite 原子写入与自动备份
- 本地代理、自动故障转移、熔断器
- Windows · macOS · Linux（Tauri 2）

---

## 支持的工具

Claude Code · Claude Desktop · Codex · Gemini CLI · OpenCode · OpenClaw · Hermes

---

## 升级说明

此为初始版本，无旧版 OpenSunstar 迁移路径。首次启动会将现有 CLI 配置导入为 **default** 供应商。

## 已知限制（v0.1.0）

- 除 Claude Code 外，大多数 CLI **切换供应商后需重启终端**。
- 项目阶段 / MVP 进度暂存 localStorage，后续版本计划迁入 SQLite。
- WebDAV / S3 后台自动同步错误在设置页内联展示，暂无全局 Toast。
