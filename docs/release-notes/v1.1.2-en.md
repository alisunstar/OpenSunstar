# OpenSunstar v1.1.2 Release Notes

**Release date:** 2026-07-07  
**License:** MIT  
**Database schema:** v28

---

## Overview

v1.1.2 continues the **1.1.x** release line (after v1.1.0 / v1.1.1) and closes reliability gaps from the early 0.x period: provider management is fully reachable, portfolio stage/progress persists in SQLite, AI insight keys use the OS keychain, and documentation dead links are fixed.

**Download:** [GitHub Releases](https://github.com/alisunstar/OpenSunstar/releases/latest)

---

## Highlights

### Provider management restored

- Quick Start: **Quick Connect | Manage Providers** tabs
- Edit, delete, failover queue via `ExpertProviderPanel`
- Success screen no longer misroutes to proxy settings

### Portfolio data reliability

- Project **stage** and **MVP progress** in SQLite (schema v28)
- One-time migration from `localStorage`

### Security & navigation

- AI insight API keys in **OS Keychain**
- Sidebar **Sync & Backup** entry

### Versioning

- App SemVer aligned with tags: **1.1.2**
- See [VERSIONING.md](../VERSIONING.md); `get_build_info` returns app + schema version
