# OpenSunstar v1.1.2 发布说明

**发布日期：** 2026-07-08  
**许可证：** MIT  
**数据库 schema：** v28

---

## 概述

v1.1.2 延续 **1.1.x** 发版线（接在 v1.1.0 / v1.1.1 之后），并修复 0.x 阶段遗留的可靠性问题：供应商管理恢复可达、看板阶段/进度可靠存入 SQLite、AI 洞察密钥迁入系统钥匙串、文档死链修复。

**下载：** [GitHub Releases](https://github.com/alisunstar/OpenSunstar/releases/latest)

---

## 亮点

### 供应商管理恢复

- 快速接入：**「快速接入 | 管理供应商」** Tab
- 可编辑、删除供应商，完整 `ProviderList` 与故障队列
- 接入成功页不再误跳代理设置

### 看板数据可靠性

- 项目 **阶段** 与 **MVP 进度** 存入 SQLite（schema v28）
- 首次启动从 `localStorage` 自动迁移

### 安全与导航

- AI 洞察 API Key 存入 **OS Keychain**
- 侧栏新增 **「同步备份」** 入口

### 版本治理

- 应用版本与 Git tag 对齐：**1.1.2**
- 详见 [VERSIONING.md](../VERSIONING.md)；`get_build_info` 返回双版本号

---

## 升级说明

| 自版本 | 说明 |
|--------|------|

| **v1.1.0 / v1.1.1** | 就地升级；若尚未到 schema v28 则自动迁移 |
