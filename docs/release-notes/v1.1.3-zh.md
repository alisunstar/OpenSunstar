**下载：** [GitHub Releases](https://github.com/alisunstar/OpenSunstar/releases/latest)

## 功能优化

- **方法论与编排**：侧边栏「配置维度」更名为「方法论与编排」，命名更贴近实际能力
- **方法论框架 Tab**：进入页面自动恢复上次扫描结果，无需每次手动点「扫描所有项目」
- **方法论框架 Tab**：检测到框架后，卡片展示「去预设编排」入口，并推荐流程档位（完整 / 标准 / MVP / 仅审查）
- **方法论框架 Tab**：未检测到的框架收进「框架参考目录」折叠区，减少信息噪音
- **方法论框架 Tab**：区分「未扫描」与「已扫描但未检测到」两种状态，避免误导
- **自定义编排（Recipe Composer）**：阶段图默认适应窗口宽度，16 寸屏全屏不再截断；支持「实际大小 / 适应宽度」切换
- **自定义编排（Recipe Composer）**：字段、占位符、图例、按钮及提示文案全面中文化（zh / en / zh-TW / ja）
- **导航精简**：移除侧栏重复的「同步备份」入口，统一在「设置 → 高级」管理（底部同步状态条保留）

## BUG 修复

- **AI 资产配置**：修复工作区「去配置」进入项目资产配置时触发 React #310 崩溃（Hooks 顺序问题）
- **方法论框架 Tab**：修复刷新或重新进入后扫描结果不恢复、卡片误显示「N 项目均未检测到」的问题
