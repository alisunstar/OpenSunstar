# OpenSunstar v0.1.0 リリースノート

**リリース日：** 2026-06-19  
**ライセンス：** MIT

---

## 概要

OpenSunstar **v0.1.0** は**初の公開リリース**です。AI コーディング CLI 向けのネイティブデスクトップコントロールプレーンとして、プロバイダー切り替え、Agent 設定（MCP / Skills / Prompts）、マルチリポジトリポートフォリオ洞察を一つのアプリで管理できます。

**ダウンロード：** [GitHub Releases](https://github.com/alisunstar/OpenSunstar/releases/tag/v0.1.0)

---

## ハイライト

### すぐに接続

- **Simple Connect** — プロバイダー → API キー → CLI 適用
- **50+ プリセット**、7 CLI ツール対応
- **トレイ**から即時切り替え（Claude Code はホットスイッチ）

### 統合 Agent ワークスペース

- MCP パネル、Skills、Prompts、コマンド、Hooks、権限、Subagent、セッション

### ポートフォリオ & 運用

- **ポートフォリオダッシュボード** — マルチ Git リポジトリコックピット
- **7 日間統一コミット指標**、AI 週次レポート
- 使用量ダッシュボード、予算アラート、WebDAV / S3 同期

### 信頼性

- SQLite アトミック書き込み、自動バックアップ
- ローカルプロキシ、フェイルオーバー、サーキットブレーカー
- Windows · macOS · Linux（Tauri 2）

---

Claude Code · Claude Desktop · Codex · Gemini CLI · OpenCode · OpenClaw · Hermes

## アップグレード

初回リリースのため、旧バージョンからの移行はありません。初回起動時に既存 CLI 設定が **default** プロバイダーとしてインポートされます。
