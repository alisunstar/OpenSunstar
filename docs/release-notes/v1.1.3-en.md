# OpenSunstar v1.1.3 Release Notes

**Release date:** 2026-07-08  
**License:** MIT

## Improvements

- **Methodology & Orchestration:** Renamed sidebar entry from「Configuration Dimensions」to「Methodology & Orchestration」
- **Methodology framework tab:** Auto-restore saved SDD scan results on page open—no need to rescan every visit
- **Methodology framework tab:** Detected frameworks show a「Go to Preset Orchestration」CTA with recommended preset tier (Full / Standard / MVP / Review-only)
- **Methodology framework tab:** Undetected frameworks moved into a collapsible reference catalog; clearer「Not scanned」vs「Scanned but absent」states
- **Recipe Composer:** Stage graph fits container width by default (no cut-off on smaller screens); toggle between fit-width and actual size
- **Recipe Composer:** Full UI localization for labels, placeholders, legend, buttons, and toasts (zh / en / zh-TW / ja)
- **Navigation:** Removed duplicate「Sync & Backup」sidebar entry; sync/backup remains under Settings → Advanced (status bar unchanged)

## Bug Fixes

- **Project AI assets:** Fixed React #310 crash when opening「Configure」from workspace (Hooks order in ProjectAssetPanel)
- **Methodology framework tab:** Fixed scan results not restoring after refresh and incorrect「all projects absent」badges before scan
